# OpenEye Site Designer

OpenEye Site Designer is an internal planning tool for laying out camera coverage on imported floor plans, site maps, and aerial images.

The tool allows users to calibrate scale, place camera models, adjust mounting details, visualize field of view, estimate pixel density, and export design summaries for project planning.

> Internal planning tool. Final camera placement, mounting, and performance should be verified in the field.

---

## Features

- Import floor plans, site surveys, or aerial images
- Calibrate map scale using two known points
- Place OpenEye camera models onto a map
- Adjust camera height, lens, tilt, rotation, and coverage
- Visualize camera field of view
- Calculate pixel density / PPF by distance
- Show coverage quality zones
- Identify blind spots based on camera height and tilt
- Measure distances directly on the map
- Add multiple floors to a project
- Export floor or full-project design summaries
- Review selected camera details in the inspector panel
- Access camera spec sheet links where available

---

## Intended Use

This tool is designed to help with early-stage video system planning, sales engineering, quoting, and customer conversations.

It can help estimate:

- Camera placement
- Coverage range
- Pixel density
- Blind spots
- Camera counts
- Floor-by-floor layouts
- Design assumptions

## Basic Workflow

1. Import a map or floor plan
2. Set the scale using two known points
3. Select a camera from the library
4. Place the camera on the map
5. Adjust height, lens, tilt, and rotation
6. Review coverage and PPF estimates
7. Repeat for each camera
8. Export the floor or full project summary

---

## Design Notes

Calculations are based on user-provided scale, camera specifications, mounting height, field of view, lens settings, and camera angle.

Accuracy depends on:

- Correct map scale
- Accurate mounting height
- Correct camera model data
- Real-world mounting conditions
- Obstructions not shown on the map
- Lighting and environmental conditions

All outputs should be treated as planning estimates.
